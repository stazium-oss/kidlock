# Настройка Tailscale для удалённого доступа

## Что такое Tailscale

Tailscale создаёт зашифрованный туннель между твоими устройствами.
После настройки ты можешь открыть веб-панель KidLock с телефона или
ноутбука из любой точки мира — как будто ты в домашней сети.

Бесплатно до 100 устройств. Роутер трогать не нужно.

---

## Установка (5 минут)

### Шаг 1 — На домашнем ПК (где стоит KidLock)

1. Перейди на **tailscale.com/download**
2. Скачай Tailscale для Windows
3. Установи и войди в аккаунт (Google или GitHub)
4. Tailscale появится в трее — статус «Connected»
5. Запомни IP-адрес: в трее → правый клик → «This device» — там будет адрес вида `100.x.x.x`

### Шаг 2 — На твоём телефоне / ноутбуке

1. Установи Tailscale из App Store / Google Play / tailscale.com
2. Войди в **тот же аккаунт**
3. Устройства увидят друг друга

### Шаг 3 — Проверь

На телефоне открой браузер и перейди по адресу:

```
http://100.x.x.x:8000
```

(вместо `100.x.x.x` — IP-адрес домашнего ПК из шага 1)

Должна открыться веб-панель KidLock.

---

## Постоянный адрес (опционально)

Чтобы не запоминать IP, можно назначить имя устройству:

1. Открой **login.tailscale.com**
2. Найди домашний ПК → «Edit» → задай имя, например `kidlock-pc`
3. Теперь адрес: `http://kidlock-pc:8000`

---

## Безопасность

- Трафик зашифрован (WireGuard)
- Доступ только с устройств в твоём Tailscale-аккаунте
- Пароль веб-панели задаётся в `server/secrets.json`

---

## Изменение пароля веб-панели

Открой файл `server/secrets.json`:

```json
{
  "parent_password": "твой_новый_пароль",
  "jwt_secret":      "случайная_строка_не_менее_32_символов"
}
```

Перезапусти сервер после изменения.

**Важно:** измени `jwt_secret` на случайную строку — это ключ шифрования токенов.
Можно сгенерировать: `python -c "import secrets; print(secrets.token_hex(32))"`
